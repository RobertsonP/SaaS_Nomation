## üîç TROUBLESHOOTING GUIDE

### "Build fails with TypeScript errors"
```
1. Read error messages carefully
2. Locate the file and line number
3. Check for:
   - Missing type definitions
   - Incorrect imports
   - Undefined variables
   - Type mismatches
4. Fix errors one by one
5. Run npm run build again
6. Repeat until clean build
```

### "Tests aren't executing"
```
Check:
1. Is dev server running?
2. Are selectors valid? (check execution.service.ts)
3. Are there errors in browser console?
4. Is database connected?
5. Check test execution logs
```

### "Frontend not connecting to backend"
```
Check:
1. Is backend running? (npm run dev in /backend)
2. Is frontend running? (npm run dev in /frontend)
3. Correct API URLs in frontend/src/lib/api.ts?
4. CORS configured correctly?
5. Check browser network tab for 404/500 errors
```
